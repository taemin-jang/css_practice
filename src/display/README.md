## block

- 항상 좌 상단부터 쌓임

## inline

- 우측으로 쌓임
- width, height 값이 적용이 안됨

## inline-block

- 우측으로 쌓이면서 width, height 값이 적용

## none

- 요소를 화면상에서 제거할 경우 사용
